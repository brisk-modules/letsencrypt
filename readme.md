# Brisk: Let's Encrypt

An extension for [Brisk](http://github.com/makesites/brisk) that makes it easy to authenticate an SSL certificate using Let's Encrypt.


## Dependencies

* [Brisk](https://github.com/makesites/brisk)


## Install

Using npm:
```
npm install brisk-letsencrypt
```
